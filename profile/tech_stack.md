# Tech Stack

## Primary Stack

- **Languages & Frameworks:** TypeScript (primary), React, Next.js, Remix, React Router, Vite, RxJS, MobX, React Native, Tailwind, CSS Modules, Canvas, WebSockets, WebRTC, Playwright, Cypress, Webpack
- **Blockchain:** EVM (Ethereum), Solana, Tezos, Aptos, Flow, wagmi, viem, ethers.js, web3.js, smart contract integrations (consumer side only — no Solidity), blockchain indexers (ponder.sh, The Graph), IPFS
- **Backend & Infra:** Node.js, Bun, Prisma, Drizzle, Express.js, Hono.js, Deno, PostgreSQL, MongoDB, SSR, Backend-for-Frontend, REST API design, GraphQL, tRPC, Docker, Railway, Supabase, AWS (Lambda, S3, media), CI/CD, open-source SDK development

## Engineering Focus

- **Primary strength: Frontend** — UI architecture, design systems, component libraries, performance, UX polish
- **Secondary strength: Backend** — TypeScript-only (Node.js / Bun); full backend services, REST APIs, BFF layers, infrastructure
- For fullstack roles, expect more frontend ownership than backend

## Hard Limits (filter these out)

- **Backend language:** TypeScript only (Node.js or Bun). Does NOT use Rust, Python, Go, or other backend languages.
- **Smart contracts:** Does NOT write Solidity or Rust/Anchor. Blockchain experience is on the integration/consumer side — wallets, SDKs, indexers, dApps.
- Filter out roles that require Solidity/smart contract development or backend work in non-TypeScript languages as a primary requirement.
