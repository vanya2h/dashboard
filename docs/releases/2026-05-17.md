# Release 2026-05-17

## Packages

| Package | Version |
| ------- | ------- |
| `web`   | `0.1.2` |

## Highlights

This release completes the product rename from "Learning Tracker" to **Sheafu** with a new SVG logo component and a dedicated display typeface (Host Grotesk). The CI pipeline was also fixed to prevent redundant build triggers when locale files are updated during release.

## Changes

### New Features

- **Sheafu logo**: New `SheafuLogo` SVG component with separate light and dark gradient paths (orange→pink→blue). Rendered in the header (32px) and footer (16px). — `packages/web/src/components/SheafuLogo.tsx`
- **Display font**: Host Grotesk Variable added as `--font-display` / `font-display` Tailwind utility. Applied to the site name in the header and the hero heading on the dashboard. — `packages/web/src/index.css`, `packages/web/src/components/Header.tsx`, `packages/web/src/components/Dashboard.tsx`
- **Branding update**: Header wordmark changed to bold + `font-display`; footer copyright updated to "Sheafu" with logo inline. — `packages/web/src/components/Header.tsx`, `packages/web/src/components/Footer.tsx`

### Bug Fixes

- Header backdrop blur now only activates when scrolled **and** the theme is dark, preventing the frosted-glass effect on light mode. — `packages/web/src/components/Header.tsx:84`
- Release CI no longer auto-merges version bumps into the `develop` branch, removing a source of unexpected merge conflicts. — `.github/workflows/rc-release.yml`
- Locale extraction commit in the release workflow now appends `[skip ci]` to avoid triggering a redundant pipeline run. — `.github/workflows/release.yml`

### Dependencies

- Added `@fontsource-variable/host-grotesk` for the display typeface.
